\# FixPoint - Maintenance Management System



Web-based maintenance request management system for Saudi Electronic University.



\## Features

\- User registration and authentication

\- Submit maintenance requests with photos

\- Request tracking and status updates

\- Admin dashboard and management

\- Technician task assignment

\- User feedback system



\## Technology Stack

\- \*\*Frontend:\*\* HTML5, CSS3, JavaScript

\- \*\*Backend:\*\* PHP 7.4+

\- \*\*Database:\*\* MySQL 8.0

\- \*\*Server:\*\* Apache (XAMPP)



\## Installation



1\. Clone the repository

2\. Import `fixpoint.sql` to MySQL

3\. Configure `config/database.php`

4\. Access: http://localhost/fixpoint
